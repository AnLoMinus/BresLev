# 🌐 אתר ספר המידות

אתר אינטראקטיבי לספר המידות מאת רבי נחמן מברסלב.

## ✨ תכונות

- 📚 **93 פרקים מלאים** — כל פרקי ספר המידות
- 🔍 **חיפוש מתקדם** — חפש לפי שם פרק או מספר
- 📱 **רספונסיבי** — עובד מצוין על כל המכשירים
- ⚡ **טעינה דינמית** — הפרקים נטענים לפי דרישה
- 🎨 **עיצוב מודרני** — ממשק נקי ויפה

## 🚀 התקנה והרצה

### אפשרות 1: שרת מקומי פשוט

```bash
# התקן Python (אם אין)
# ואז הרץ:

cd website
python3 -m http.server 8000

# או עם Node.js:
npx http-server -p 8000
```

ואז פתח בדפדפן: `http://localhost:8000`

### אפשרות 2: עם Live Server (VS Code)

1. התקן את התוסף "Live Server" ב-VS Code
2. לחץ ימני על `index.html`
3. בחר "Open with Live Server"

## 📁 מבנה הקבצים

```
website/
├── index.html          # עמוד ראשי
├── style.css          # עיצוב
├── app.js             # לוגיקה ראשית
├── chapters-loader.js # טעינת פרקים
└── README.md          # קובץ זה
```

## 🔧 איך זה עובד?

1. **chapters-loader.js** — טוען את רשימת כל הפרקים
2. **app.js** — מנהל את האפליקציה, חיפוש, וניווט
3. הפרקים נטענים דינמית מתיקיית `../SparMidot/chapters/`
4. Markdown מומר ל-HTML באמצעות `marked.js`

## 📝 הערות

- האתר משתמש ב-`marked.js` להמרת Markdown ל-HTML
- הפרקים נטענים דרך `fetch()` - דורש שרת (לא יעבוד עם `file://`)
- כל הקישורים הפנימיים בין פרקים מתוקנים אוטומטית

## 🎨 התאמה אישית

ניתן לשנות את הצבעים ב-`style.css`:

```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
}
```
