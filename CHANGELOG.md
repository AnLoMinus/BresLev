## [0.6.0] — כ״ב בחשוון תשפ״ו (22.11.2025)

### 🌐 יצירת אתר אינטראקטיבי לספר המידות

#### ✨ מה נוצר

- **אתר מלא** בתיקיית `/website/`
- **index.html** — עמוד ראשי עם ממשק מודרני
- **style.css** — עיצוב רספונסיבי ומודרני (7777 שורות)
- **app.js** — לוגיקה ראשית של האפליקציה
- **chapters-loader.js** — סקריפט לטעינת כל הפרקים דינמית
- **README.md** — הוראות שימוש והתקנה

#### 🎨 תכונות האתר

- 📚 **93 פרקים מלאים** — כל פרקי ספר המידות נטענים דינמית
- 🔍 **חיפוש מתקדם** — חפש לפי שם פרק או מספר
- 📱 **רספונסיבי** — עובד מצוין על כל המכשירים (mobile-first)
- ⚡ **טעינה דינמית** — הפרקים נטענים לפי דרישה (lazy loading)
- 🎨 **עיצוב מודרני** — ממשק נקי ויפה עם אנימציות
- 🔗 **ניווט פנימי** — קישורים בין פרקים עובדים אוטומטית
- 📖 **Markdown rendering** — שימוש ב-marked.js להמרת MD ל-HTML
- 🎯 **URL routing** — תמיכה ב-hash navigation (#chapter-1)

#### 🔧 טכנולוגיות

- HTML5 + CSS3 (Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (ללא frameworks)
- marked.js להמרת Markdown
- Fetch API לטעינת קבצים דינמית

#### 📁 מבנה האתר

```
website/
├── index.html          # עמוד ראשי (2851 bytes)
├── style.css          # עיצוב מלא (7777 bytes)
├── app.js             # לוגיקה ראשית (7673 bytes)
├── chapters-loader.js # טעינת פרקים (7950 bytes)
└── README.md          # הוראות שימוש (2297 bytes)
```

#### 🚀 איך להריץ

```bash
cd website
python3 -m http.server 8000
# או
npx http-server -p 8000
```
